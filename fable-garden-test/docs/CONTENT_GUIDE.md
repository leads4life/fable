# Content guide
Add a record to `js/data.js` with ID, title, slug, alternate titles, description, characters, animals, themes, keywords, reading time, story paragraphs, moral, short moral, moral explanation, reflection questions, source note, publication date, and modification date.

Write original wording: modern translations can be copyrighted. Create an HTML page with correct relative paths, unique title/description/canonical metadata, and a matching data ID. Update internal links and `sitemap.xml`; never duplicate another page’s metadata.
