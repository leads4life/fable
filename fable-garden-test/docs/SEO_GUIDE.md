# SEO guide
Every indexable page uses a distinct title, meta description, canonical URL, Open Graph metadata, heading hierarchy, internal links, and descriptive image alt text. The homepage has WebSite, Organization, and FAQ schema; the fable page has Article and BreadcrumbList schema. `robots.txt` and `sitemap.xml` use the example domain.

## Launch checklist
- Replace `https://example.com` in config, canonical tags, schemas, sitemap, robots, and Open Graph URLs.
- Update publication dates and sitemap lastmod values.
- Deploy to static hosting, verify canonical URLs, and submit the sitemap.
- Give each added fable unique metadata and avoid duplicate pages.
